What is Simulation?

  • To simulate is to imitate. Thus, when a simulation is performed, its goal is to mimic a specified event or process. Such simulations are best run on computers. The better such simulations recreates the behavior, the more the results can be trusted and used in real-life situations.

Why Use Simulation?

  • Offers risk-free testing of extreme-use cases.

  • Saves labor, time, and resources.

  • Provides accurate predictions of real-life performance.

  • Allows for repeatable testing.

  • Optimizes quickly for the best results.

  • Provides useful animations and visuals for stakeholders.

When to Use Simulation?

  • Testing on the real thing is not always realistic, so simulating can greatly reduce total cost, build time, and labor before implementing plans in the real-world.

How to Build a Simulation?

  1. Define the model's inputs and outputs.

  2. Determine model behavior.

  3. Construct the model.

  4. Verify performance. Do the outputs make sense for the given input? If not, verify steps 1-4, make necessary changes, and repeat.

Examples of Simulation

Material Handling Equipment Simulation at Dematic

  • Building large-scale material handling equipment is expensive. Those that need it installed want to be sure that their new equipment will perform as best as possible. Since building and testing the equipment would be expensive and interrupt current operations, simulation must be done instead to test and ensure the performance of the new system.

  • Simulation of material handling is usually done with discrete-event simulation.

Orbital Mechanics

  • Getting a rocket to get where it's wanted is complicated. After accounting for aerodynamic drag, change in weight during burns, gravity from nearby moons, solar wind, or even having an engine not burn when lit, finding a direct mathematical equation that describes its entire motion is basically impossible. Therefore, simulations must be run to determine where a rocket ends up for given burn times, lengths, strengths, and directions.

  • All these parameters must then be optimized to ensure that it ends up where it was planned, has enough fuel to get there, enters the atmosphere at a survivable speed and angle, and lands at survivable speeds too. Simulations are used to solve these problems, and must research and development has gone into making these faster, more accurate, and more robust.

Simulating Stable Canard Flight

  • A jet with forward-swept wings has its advantages, however, this setup is so aerodynamically unstable that a pilot doesn't have the reaction speed to keep it steady. Only an active electronic feedback loop is fast enough to keep the jet from spinning uncontrollably in the sky. Therefore, developing this feedback loop requires a model that can be simulated.

  • The simulation would compute how the jet would change pitch angle over time given its many input variables. If the simulation shows that the jet becomes unstable, then the model controlling the electronics need to be tuned better. If it shows excellent stability, then such electronics used in the simulation should be used to build the real jet.
